# `type`

**References:**
- https://code.kx.com/q/ref/type/

Datatype of an object.

-----------------------------------------------------------------------------------------------------
-----------------------------------------------------------------------------------------------------

## Syntax

~~~~
type x    type[x]
~~~~

where
- `x` is any object


Returns:
datatype of `x`


The datatype is a `short int`:
- negative for atoms
- positive for vectors
- 0h for a general list.

-----------------------------------------------------------------------------------------------------
-----------------------------------------------------------------------------------------------------

## Examples


~~~~
type 5
-6h
~~~~

~~~~
type 2 3 5
6h
~~~~


~~~~
type (2;3 5;"hello")
0h
~~~~

~~~~
type (+)
102h
~~~~
