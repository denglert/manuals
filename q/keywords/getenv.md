# `getenv`

**References:**
- https://code.kx.com/q/ref/getenv/

Get the value of an environment variable.

-------------------------------------------------------------------------------------------------------
-------------------------------------------------------------------------------------------------------

## Syntax

~~~~
getenv x
~~~~

or


~~~~
getenv[x]
~~~~


where
- `x` is a symbol atom naming an environment variable

**Returns:**
- Value of the environment variable

-------------------------------------------------------------------------------------------------------

### Examples

~~~~
getenv`QHOME
~~~~
