# `show` 

**References:**
- https://code.kx.com/q/ref/show/

Format and display at the console.

-----------------------------------------------------------------------------------------------------
-----------------------------------------------------------------------------------------------------

## Syntax

~~~~
show x, show[x]
~~~~


Formats `x` and writes it to the console; returns the identity function `(::)`.

-----------------------------------------------------------------------------------------------------
-----------------------------------------------------------------------------------------------------

## Examples


~~~~
q)f:{show a:x<5; sum a} 
q)f 2 3 5 7 3
11001b
3
~~~~
