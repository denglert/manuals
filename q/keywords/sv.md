# `sv`

**References:**
- https://code.kx.com/q/ref/sv/

''Scalar from vector''

join strings, symbols, or filepath elements
decode a vector to an atom

---------------------------------------------------------------------------------------
---------------------------------------------------------------------------------------

## Syntax

~~~~
x sv y    sv[x;y]
~~~~

---------------------------------------------------------------------------------------
---------------------------------------------------------------------------------------

## Examples


### Join

~~~~
"," sv ("one";"two";"three")
"one,two,three"
~~~~

~~~~
"\t" sv ("one";"two";"three")
"one\ttwo\tthree"
~~~~
