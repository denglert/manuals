# Creating a table in `q`


-------------------------------------------------------------------------------------------------------------------------
-------------------------------------------------------------------------------------------------------------------------


## Creating a table directly


**Example(s):**


~~~~
dates  : 2021.01.01, 2021.01.01, 2021.01.02
qtys   : 23, 25, -12
prices : 0.3, 0.1, 0.9

table: ([]
        date  : dates;
        qty   : qtys;
        price : prices 
       )
~~~~

-------------------------------------------------------------------------------------------------------------------------
-------------------------------------------------------------------------------------------------------------------------

## Creating a table via dictionary

**Example(s):**

~~~~
dates  : 2021.01.01, 2021.01.01, 2021.01.02
qtys   : 23, 25, -12
prices : 0.3, 0.1, 0.9


dict : (`date; `qty; `price) !
       (dates;qtys;prices)

table: flip dict
~~~~
