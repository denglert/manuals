# Casting in `q`

**References:**
- https://code.kx.com/q/ref/cast/

---------------------------------------------------------------------------------------------------------------
---------------------------------------------------------------------------------------------------------------

## Examples

Int to float:

~~~~
q) `float$1
1.0
~~~~

Float to long int:

~~~~
q) `long$1.0
1
~~~~


Int to boolean:

~~~~
q) `booleans$1
1b
~~~~


Int to date:

~~~~
q) `date$31
2000.02.01
~~~~
