# Linear Algebra in `q`


## Element-wise operations

~~~~
A: (1 2f; 3 4f);
B: (-2 1f; -3 1f);
~~~~


### Addition

~~~~
A + B
~~~~

### Subtraction

~~~~
A - B
~~~~


### Multiplication

~~~~
A * B
~~~~


### Division

~~~~
A % B
~~~~

## Composite operations


### Matrix multiply

**References:**
- https://code.kx.com/q/ref/mmu/


~~~~
A $ B
~~~~

~~~~
A mmu B
~~~~

### Matrix inverse

**References:**
- https://code.kx.com/q/ref/inv/

~~~~
inv A
