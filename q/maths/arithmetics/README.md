# Arithmetics in `q`


## Addition

Operator: `+`


**Example(s):**


~~~~
q)3+4
7
~~~~

-----------------------------------------------------------------------------------
-----------------------------------------------------------------------------------

## Subtraction

Operator: `-`

**Example(s):**

~~~~
q)8-3
5
~~~~


-----------------------------------------------------------------------------------
-----------------------------------------------------------------------------------

## Multiplication


Operator: `*`

**Example(s):**

~~~~
q)3*4
12
~~~~


-----------------------------------------------------------------------------------
-----------------------------------------------------------------------------------


## Division


Operator: `%`


**Note:**
- Division always returns a float response.


**Example(s):**

~~~~
q)8%4
2f
~~~~
