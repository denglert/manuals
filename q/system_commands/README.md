# System commands


**References:**
- https://code.kx.com/q/basics/syscmds/

**System commands:**
- [`\a`](#backslasha) tables
- [`\b`](#backslashb) views
- `\B` pending views
- `\c` console size
- `\cd` change directory
- `\C` HTTP size
- [`\d`](#backslashd) directory
- `\e` error trap client
- `\E` TLS server mode
- [`\f`](#backslashf) functions
- `\g` garbage collection mode
- [`\l`](#backslashl) load file or directory 
- `\o` offset from UTC
- [`\p`](#backslashp) listening port
- [`\P`](#backslashP) precision
- `\r` replication master
- `\r` rename
- `\s` number of secondary threadss
- `\S` random seed
- `\t` timer
- `\T` timeout
- `\ts` time and space
- `\u` reload user password file
- `\v` variables
- `\w` workspace
- `\W` week offset
- `\x` expunge
- `\z` date parsing
- `\1` & `\2`  redirect
- `\_` hide q code
- `\ ` terminate
- `\`  toggle q/k
- `\\` quit
