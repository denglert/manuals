# `.Q.w` (memory stats)

**References:**
- https://code.kx.com/q/ref/dotq/#qw-memory-stats

Returns the memory stats from `\w` into a more readable dictionary.

-----------------------------------------------------------------------------------

## Syntax

~~~~
.Q.w[]
~~~~

-----------------------------------------------------------------------------------

## Example(s)

~~~~
q).Q.w[]
used| 168304
heap| 67108864
peak| 67108864
wmax| 0
mmap| 0
mphy| 8589934592
syms| 577
symw| 25436
~~~~
