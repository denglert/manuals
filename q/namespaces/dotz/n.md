# `.z.n` (UTC timespan)

System UTC time as timespan (`-16h`) in nanoseconds.

## Syntax

~~~~
.z.n
~~~~

------------------------------------------------------------------------------------------------------
------------------------------------------------------------------------------------------------------

## Examples

~~~~
.z.n
0D14:19:12.831429000
~~~~

# `.z.N` (local timespan)

System local time as timespan (`-16h`) in nanoseconds.

## Syntax

~~~~
.z.N
~~~~

## Examples

~~~~
.z.N
0D15:19:07.049736000
~~~~
